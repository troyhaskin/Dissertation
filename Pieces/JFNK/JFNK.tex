\chapter[JFNK Solver]{Jacobian-Free Newton-Krylov Solver}

A Jacobian-Free Newton-Krylov (JFNK) solver is a generic name for a method that solves a system of equations, which are typically nonlinear.
As implied by the name, these solvers are defined by two main characteristics: lack of a need to form the exact Jacobian (Jacobian-free) and Newton-like updates solved via a Krylov method (Newton-Krylov).
